# Jogo RPG

Projeto desenvolvido em Java via CLI com o objetivo de praticar POO.

## Funcionalidades

- Escolha entre três classes:
    - Cavaleiro
    - Arqueiro
    - Mago
- Sistema de batalha por turnos
- Inimigo com ações aleatórias
- Ataques críticos
- Sistema de defesa (Parry)
- Sistema de mana e cura

## Conceitos utilizados

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Sobrescrita de métodos (Override)
- Random
- Estruturas de repetição e decisão
