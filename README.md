# Jogo RPG

Projeto desenvolvido em Java via CLI com o objetivo de praticar POO.

## Funcionalidades

- Escolha entre três classes:
    - Cavaleiro
    - Arqueiro
    - Mago
- Sistema de batalha por turnos
- Inimigo com ações aleatórias
- Ataques críticos
- Sistema de defesa (Parry)
- Sistema de mana e cura

## Conceitos utilizados

- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Sobrescrita de métodos (Override)
- Random
- Estruturas de repetição e decisão

## Como executar

1. Clone este repositório:

```bash
git clone https://github.com/asousa-dev/jogoRpg.git
```

2. Abra o projeto em uma IDE Java

3. Certifique-se de que o JDK esteja instalado

4. Execute a classe `Main.java`.

5. Siga as instruções exibidas no console para jogar.
